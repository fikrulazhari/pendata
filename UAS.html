Breast Cancer Wisconsin (Diagnostic)#

Dalam dunia machine learning, Dataset Breast Cancer Wisconsin (Diagnostic) adalah studi kasus fundamental untuk tugas klasifikasi medis. Dataset ini menyediakan fitur-fitur numerik dari sampel sel pasien yang diperoleh melalui aspirasi jarum halus (FNA). Tujuannya adalah untuk membangun model prediktif yang mampu membedakan secara akurat antara diagnosis ganas (malignant) dan jinak (benign).

Pengambilan data dari Repositori UCI untuk analisis lebih lanjut difasilitasi melalui sebuah pustaka (package) Python khusus. Untuk menggunakannya, pengguna perlu menginstal pustaka tersebut terlebih dahulu. Petunjuk instalasi dan cuplikan kode untuk mengimpor data tersedia langsung di halaman setiap dataset di situs UCI, tepatnya di bawah tombol ‚ÄúImport in Python‚Äù. Dengan mengeksekusi kode tersebut, dataset akan diunduh dan dimuat ke dalam lingkungan kerja Python, sehingga siap untuk diolah.

Sebagai ilustrasi, perintah untuk mengunduh dataset terkait dapat dilihat pada gambar dan kode berikut:

deteksi outlier#

Q1 = X.quantile(0.25)
Q3 = X.quantile(0.75)
IQR = Q3 - Q1

# Cek kondisi outlier
outliers_iqr = (X < (Q1 - 1.5 * IQR)) | (X > (Q3 + 1.5 * IQR))

# Hitung jumlah outlier per kolom
print("Jumlah outlier per fitur (IQR):")
print(outliers_iqr.sum())

# Jumlah total baris yang mengandung outlier
print(f"\nJumlah baris yang mengandung outlier (IQR): {outliers_iqr.any(axis=1).sum()}")

from ucimlrepo import fetch_ucirepo
import pandas as pd
from sklearn.preprocessing import MinMaxScaler, StandardScaler

# Ambil dataset
dataset = fetch_ucirepo(id=15)
X = dataset.data.features.copy()

# Tangani nilai '?' di 'Bare_nuclei' jika masih object
if X['Bare_nuclei'].dtype == 'object':
    X['Bare_nuclei'] = pd.to_numeric(X['Bare_nuclei'], errors='coerce')

# Tangani missing value (drop atau imputasi)
X = X.dropna()  # atau gunakan imputasi jika tidak ingin buang data

# Normalisasi dengan MinMaxScaler (0‚Äì1)
minmax_scaler = MinMaxScaler()
X_minmax = pd.DataFrame(minmax_scaler.fit_transform(X), columns=X.columns)

# Normalisasi dengan StandardScaler (mean=0, std=1)
standard_scaler = StandardScaler()
X_standard = pd.DataFrame(standard_scaler.fit_transform(X), columns=X.columns)

# Tampilkan contoh hasil normalisasi
